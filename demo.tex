\subsection{Theme Overview}
\begin{frame}{What is New in This Theme?}
    This enhanced NJU Beamer template brings a fresh, modern design while preserving the classic purple-and-gold identity of Nanjing University.
    
    \begin{block}{Key Enhancements}
        \begin{itemize}
            \item \textbf{Refined Color Palette:} Deeper purple tones with elegant gold accents and semantic colors
            \item \textbf{Modern Block Design:} Left-border accent style with subtle shadows
            \item \textbf{Enhanced Typography:} Better hierarchy and spacing throughout
            \item \textbf{Decorative Elements:} Gold accent lines and refined section transitions
        \end{itemize}
    \end{block}
    
    \begin{alertblock}{Important Note}
        This template is unofficial and created for academic and research presentations at NJU.
    \end{alertblock}
\end{frame}

\subsection{Design Philosophy}
\begin{frame}{Design Principles}
    The enhanced theme follows these core design principles:
    
    \begin{enumerate}
        \item \textbf{Contrast:} Clear distinction between text, background, and headings
        \item \textbf{Hierarchy:} Visual importance conveyed through size, weight, and color
        \item \textbf{Whitespace:} Generous margins and line spacing for readability
        \item \textbf{Consistency:} Unified visual language across all slide elements
    \end{enumerate}
    
    \vspace{0.5cm}
    \begin{exampleblock}{Typography}
        Uses Times New Roman math fonts with professional font encoding for a clean academic look.
    \end{exampleblock}
\end{frame}

\begin{frame}{Mathematical Content}
    The template handles mathematical notation elegantly:
    
    \vspace{0.3cm}
    The fundamental equation for neural networks is forward propagation:
    \[
    a^{(l)} = \sigma(W^{(l)}a^{(l-1)} + b^{(l)})
    \]
    where $\sigma$ is the activation function, $W^{(l)}$ are weights, and $b^{(l)}$ are biases at layer $l$.
    
    \vspace{0.3cm}
    \begin{block}{Key Algorithms}
        \begin{itemize}
            \item Backpropagation uses gradient descent to optimize parameters
            \item Convolutional Neural Networks excel in image processing
            \item Recurrent Neural Networks handle sequential data effectively
        \end{itemize}
    \end{block}
\end{frame}
